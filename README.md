<p align="center">

## Features
 - **Authentication:** Secure user registration and sign-in integrated via Google.
 - **User Dashboard:** Displaying the user's identity, performance stats, and comprehensive singing history.
 - **Leaderboard:** Ranks users based on their performance scores.
 - **Track List:**  Searchable library for easy song selection.
 - **Saved / Favorites:** Favorite songs can be saved, and can be viewed in the user's dashboard.
 - **Singing Session:** The primary karaoke interface featuring synchronized lyrics and a real-time pitch tracking visualizer.
 - **Karaoke Chat:** A global chat room for community interaction via Sendbird API.

## Tech Stack
 - **Frontend:** Next.js, Zod, SWR, SendBird
 - **Backend:**  Go, Fiber
 - **Karaoke Generator:** python, whisper, ffmpeg, karaoke-gen, crepe
 - **External APIs:** Google OAuth, SendBird API
