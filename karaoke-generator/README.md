# 🎤 Karaoke Pipeline
This repository contains the source code and workflow for generating high-quality karaoke assets and metadata. It automates the transition from a raw YouTube video to a synchronized lyric and pitch-mapped dataset.

## 🗺 Roadmap
Audio Retrieval: Download the highest quality audio via yt-dlp.

Source Separation: Extract instrumental and vocal stems using karaoke-gen.

Lyric Generation: Generate a timestamped JSON file using whisper.

Pitch Mapping: Extract high-resolution vocal frequency data using torchcrepe.

Data Alignment: Refine timestamps by snapping lyrics to actual detected pitch regions using preprocessed_karaoke.py.

##  Prerequisites
 - OS: macOS (optimized for Apple Silicon/MPS)
 - Environment: Python 3.11
 - System Tools: Homebrew, FFmpeg

## Usage 
### Installation
System Dependencies

```Bash
brew install python@3.11 ffmpeg yt-dlp
```
Create and activate virtual environment
```Bash
python3.11 -m venv venv
source venv/bin/activate
```

Upgrade base tools
```
pip install --upgrade pip setuptools wheel
```

Install core processing libraries
```
pip install torch torchaudio librosa soundfile numpy scipy
```

Install transcription and separation tools
```
pip install karaoke-gen openai-whisper
```

Install torchcrepe from source
```
pip install git+https://github.com/maxrmorrison/torchcrepe.git
```
### Execution Pipeline
Step 1: Download & Stem Generation

Fetch the audio and separate it into components.

```Bash
yt-dlp -f bestaudio -x --audio-format wav -o "song.%(ext)s" "YOUR_YOUTUBE_URL"
karaoke-gen song.wav "Artist" "Song"
```
Step 2: Pitch Extraction (extract_pitch.py)
Processes the vocal stem to create pitch.json. This script utilizes Apple Metal (MPS) for hardware acceleration.
```Bash
python extract_pitch.py
```

Step 3: Lyric Generation (whisper)
Use the Whisper CLI to generate the segment-based JSON file from the vocal stem.
```Bash
whisper "./stems/vocals.flac" --model medium --output_format json
```

Step 4: Refinement (preprocessed_karaoke.py)
Aligns the Whisper segments with the detected pitch regions to ensure lyrics are perfectly synced to the vocal activity.
```Bash
python preprocessed_karaoke.py
```

## 📂 Data Structure

| File | Type | Purpose |
| :--- | :--- | :--- |
| `instrumental.flac` | Audio | The backing track for the user to sing over. |
| `pitch.json` | Metadata | High-resolution array of time (t), frequency (f), and note (n). |
| `vocals.json` | Metadata | Raw transcription output from Whisper. |
| `lyrics.json` | Metadata | Final refined lyrics synced to pitch-detected voice regions. |

---

## ⚠️ Notes
- Hardware: The pipeline automatically detects if an Apple Silicon GPU (mps) is available; otherwise, it defaults to cpu.

- File naming may differs than in this guide so make sure to find the correct files and change the path according to that.

- File Paths: Ensure the whisper_path in preprocessed_karaoke.py matches the specific JSON filename generated by Whisper (which usually matches the name of the input vocal stem).
